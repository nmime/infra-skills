## Target

**Goal: +20% annually**

Agent runs indefinitely until stopped. Checks progress quarterly.


## Event Handling

### On Trade Event (fill/order)

1. Check what happened (TP hit? SL hit?)
2. Update portfolio tracking
3. Log result
4. If position closed ‚Üí Slot opens (but don't rush to fill)

### On Position Alert

1. +4% profit ‚Üí Enable trailing stop (4%)
2. -1.5% loss ‚Üí Review, consider early exit if trend changed
3. Report status

1. **Portfolio status report:**
